## Contenidos del Laboratorio

- Uso de Lambda: Definición de funciones anónimas para cálculos matemáticos.
- Transformación con map(): Conversión masiva de unidades (km/h a m/s y Celsius a Kelvin).
- Filtrado con filter(): Limpieza de datasets eliminando valores fuera de rango o erróneos.
- Agregación con reduce(): Consolidación de datos para obtener sumas y productos totales.
- Simulación de Big Data: Procesamiento de un dataset de 1,000,000 de registros.
- Desafío Avanzado: Simulación y análisis de 500,000 sensores industriales.

## Conclusiones Técnicas
- Optimización: Las funciones `map`, `filter` y `reduce` permiten un código más limpio y eficiente que los bucles tradicionales.
- Integridad: El filtrado previo es fundamental en ingeniería para asegurar que los cálculos se realicen sobre datos válidos.
- Escalabilidad: Python demuestra gran capacidad para manejar Big Data, siempre que se gestione correctamente la memoria y la salida por consola.
